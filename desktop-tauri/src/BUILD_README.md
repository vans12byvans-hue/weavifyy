# Wavify - Быстрая сборка

## 🚀 Быстрый старт

```bash
cd desktop
build.bat
```

Это всё! Скрипт автоматически:
1. Проверит Node.js
2. Проверит наличие всех файлов
3. Установит зависимости
4. Соберет приложение

## 📦 Результат

После сборки вы получите:
- `dist/Wavify-Setup-2.0.0.exe` - установщик (~150-200 MB)
- `dist/win-unpacked/` - распакованная версия для тестирования

## ✅ Что включено

### Все необходимое уже в desktop/
- ✅ Единый сервер (local-server-v2.js)
- ✅ Все API (YouTube, Яндекс, SoundCloud, Spotify)
- ✅ База данных (SQLite)
- ✅ Discord RPC
- ✅ Автообновление
- ✅ Все UI компоненты
- ✅ Все стили

### НЕ нужно
- ❌ Отдельный backend сервер
- ❌ Отдельный youtube-audio-server
- ❌ Telegram бот
- ❌ Дополнительные зависимости

## 🔍 Проверка перед сборкой

```bash
cd desktop
verify-build.bat
```

Проверит наличие всех критических файлов.

## 📋 Требования

- Node.js 18+ 
- npm 9+
- Windows 10/11
- ~500 MB свободного места

## 🐛 Проблемы?

### Ошибка: "Node.js not found"
Установите Node.js с https://nodejs.org/

### Ошибка: "File verification failed"
Запустите `verify-build.bat` чтобы увидеть какие файлы отсутствуют

### Ошибка: "Build failed"
1. Удалите `node_modules` и `dist`
2. Запустите `npm install`
3. Попробуйте снова
