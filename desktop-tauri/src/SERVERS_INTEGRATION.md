# Интеграция страницы "Серверы"

## Шаг 1: Добавить кнопку в сайдбар

В файле `desktop/index.html` найди секцию с кнопкой "Настройки" (около строки 154) и ПЕРЕД ней добавь:

```html
<button class="sidebar-btn" data-page="servers" title="Серверы">
  <svg width="24" height="24" viewBox="0 0 24 24" fill="currentColor">
    <path d="M20,13H4c-0.55,0-1,0.45-1,1v6c0,0.55,0.45,1,1,1h16c0.55,0,1-0.45,1-1v-6C21,13.45,20.55,13,20,13z M19,19h-2v-4h2V19z M20,3H4C3.45,3,3,3.45,3,4v6c0,0.55,0.45,1,1,1h16c0.55,0,1-0.45,1-1V4C21,3.45,20.55,3,20,3z M19,9h-2V5h2V9z"/>
  </svg>
</button>
```

## Шаг 2: Добавить HTML страницы

В файле `desktop/index.html` найди секцию `<!-- Settings Page -->` (около строки 317) и ПЕРЕД ней вставь содержимое файла `desktop/servers-page.html`.

## Шаг 3: Подключить стили

В файле `desktop/index.html` в секции `<head>` добавь:

```html
<link rel="stylesheet" href="servers-styles.css">
```

## Шаг 4: Подключить JavaScript

В файле `desktop/index.html` в конце, перед закрывающим тегом `</body>` добавь:

```html
<script src="servers-logic.js"></script>
```

## Шаг 5: Инициализировать логику

В файле `desktop/renderer.js` найди обработчик переключения страниц (где `data-page`) и добавь:

```javascript
// После переключения на страницу серверов
if (targetPage === 'servers-page') {
  if (typeof initServersPage === 'function') {
    initServersPage();
  }
}
```

## Готово!

Теперь в приложении появится новая вкладка "Серверы" с красивым интерфейсом управления:

- **Локальный сервер**: запуск/остановка/перезапуск, автозапуск
- **Discord RPC**: включение/выключение активности и прогресс бара
- **Telegram Bot**: placeholder для будущей функциональности

Стиль полностью соответствует скриншоту Discord с темной темой, переключателями и карточками.
