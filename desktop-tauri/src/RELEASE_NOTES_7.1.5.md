# Weavify 7.1.5 - Исправления и улучшения стабильности

## 🐛 Исправления

### Критические баги
- **Исправлена ошибка `favoriteBtn is not defined`** - приложение больше не падает при попытке добавить трек в избранное
- **Исправлены конфликты воспроизведения** между режимом "Моя волна" и поиском - теперь треки из поиска корректно воспроизводятся

### Стабильность
- Добавлены проверки на `null` для DOM элементов в `loadCarousel()` и `loadWaveTracks()` - предотвращает ошибки при отсутствии элементов на странице
- Улучшена обработка несоответствия состояния аудио - автоматическое возобновление воспроизведения при рассинхронизации

### Логирование
- Изменен уровень логирования для некритичных предупреждений (customization, update checker) с `ERROR` на `INFO`
- Добавлено подробное логирование для отладки воспроизведения треков из поиска

## ⚡ Улучшения

- Улучшена стабильность переключения между режимами воспроизведения (волна, поиск, плейлисты)
- Оптимизирована работа с плейлистами и очередью воспроизведения

## 📦 Установка

### Новая установка
1. Скачайте `Weavify-Setup-7.1.5.exe`
2. Запустите установщик
3. Следуйте инструкциям

### Обновление
Если у вас установлена версия 7.1.0 или новее:
- Приложение автоматически предложит обновиться при следующем запуске
- Или скачайте установщик вручную
