# 🎨 Руководство по кастомизации Weavify

## Обзор

Weavify предлагает полную систему кастомизации, позволяющую настроить внешний вид приложения под ваши предпочтения.

## Доступ к настройкам

1. Откройте приложение
2. Нажмите на кнопку "Настройки" в боковой панели (иконка шестеренки)
3. Прокрутите до раздела "Тема оформления"

## Готовые темы

### Доступные темы:

1. **Темная** (по умолчанию)
   - Черный фон
   - Белый акцент
   - Идеально для ночного использования

2. **Светлая**
   - Белый фон
   - Черный акцент
   - Для дневного использования

3. **Синяя**
   - Темный фон
   - Синий акцент (#2196F3)
   - Классический стиль

4. **Фиолетовая**
   - Темный фон
   - Фиолетовый акцент (#9C27B0)
   - Элегантный вид

5. **Зеленая**
   - Темный фон
   - Зеленый акцент (#4CAF50)
   - Природный стиль

6. **Красная**
   - Темный фон
   - Красный акцент (#f44336)
   - Энергичный вид

7. **Оранжевая**
   - Темный фон
   - Оранжевый акцент (#ff9800)
   - Теплый стиль

8. **Розовая**
   - Темный фон
   - Розовый акцент (#E91E63)
   - Яркий вид

### Как применить тему:

1. Нажмите на карточку нужной темы
2. Тема применится мгновенно
3. Настройки сохранятся автоматически

## Пользовательские цвета

### Настройка цветов:

1. **Акцентный цвет** - основной цвет интерфейса (кнопки, активные элементы)
2. **Основной фон** - цвет главного фона приложения
3. **Вторичный фон** - цвет карточек и панелей

### Как создать свою тему:

1. Нажмите на цветовой пикер для нужного параметра
2. Выберите желаемый цвет
3. Повторите для других параметров
4. Нажмите "Применить"
5. Для сброса нажмите "Сбросить"

## Анимация текстов

### Доступные анимации:

1. **Без анимации** - статичный текст
2. **Плавное появление** - текст плавно появляется
3. **Скольжение** - текст въезжает сбоку
4. **Отскок** - текст появляется с эффектом отскока

### Как изменить:

1. Выберите нужный вариант анимации
2. Изменения применятся мгновенно
3. Проверьте на странице плеера с текстами

## Размер текста лирики

### Настраиваемые параметры:

1. **Активная строка** (16-32px)
   - Текущая воспроизводимая строка
   - По умолчанию: 20px

2. **Предыдущая строка** (12-24px)
   - Строка перед текущей
   - По умолчанию: 16px

3. **Дальние строки** (10-20px)
   - Остальные строки
   - По умолчанию: 14px

### Как настроить:

1. Перетащите ползунок для нужного параметра
2. Значение обновится в реальном времени
3. Проверьте результат на странице плеера

## Прозрачность фона

### Настройка:

1. Перетащите ползунок "Прозрачность"
2. Диапазон: 0% (полностью прозрачный) - 100% (полностью непрозрачный)
3. По умолчанию: 30%
4. Влияет на видимость фонового видео

## Сохранение настроек

### Автоматическое сохранение:

- Все настройки сохраняются автоматически в localStorage
- При следующем запуске приложения настройки восстановятся
- Не требуется ручное сохранение

### Сброс настроек:

1. Для сброса темы: выберите "Темная" тема
2. Для сброса пользовательских цветов: нажмите "Сбросить"
3. Для сброса других настроек: установите значения по умолчанию

## Советы

### Рекомендации по выбору цветов:

1. **Контрастность** - выбирайте контрастные цвета для лучшей читаемости
2. **Яркость** - избегайте слишком ярких цветов для фона
3. **Гармония** - используйте цвета из одной палитры

### Рекомендации по размеру текста:

1. **Активная строка** - должна быть самой крупной
2. **Предыдущая строка** - немного меньше активной
3. **Дальние строки** - самые мелкие для фокуса на текущей

### Рекомендации по анимации:

1. **Без анимации** - для слабых компьютеров
2. **Плавное появление** - универсальный вариант
3. **Скольжение/Отскок** - для эффектного вида

## Горячие клавиши

Настройки не имеют горячих клавиш, но вы можете быстро открыть их:

1. Нажмите на иконку настроек в боковой панели
2. Или используйте навигацию через меню

## Технические детали

### Хранение данных:

- Настройки хранятся в `localStorage`
- Ключ: `weavify-theme`
- Формат: JSON

### CSS переменные:

```css
--primary-color: цвет текста
--accent-color: акцентный цвет
--bg-color: основной фон
--secondary-bg: вторичный фон
--tertiary-bg: третичный фон
--lyrics-active-size: размер активной строки
--lyrics-prev-size: размер предыдущей строки
--lyrics-far-size: размер дальних строк
--bg-opacity: прозрачность фона
```

### Data атрибуты:

```html
<body data-theme="dark" data-animation="fade">
```

## Устранение неполадок

### Настройки не сохраняются:

1. Проверьте, что localStorage не заблокирован
2. Очистите кэш браузера
3. Перезапустите приложение

### Цвета не применяются:

1. Нажмите "Применить" после выбора цветов
2. Проверьте консоль на ошибки (F12)
3. Попробуйте выбрать готовую тему

### Анимация не работает:

1. Проверьте, что выбрана анимация (не "Без анимации")
2. Откройте страницу плеера с текстами
3. Убедитесь, что трек имеет тексты
