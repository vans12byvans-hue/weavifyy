# Настройка Яндекс.Музыки

## Получение токена

1. Перейдите на сайт: https://yandex-music.readthedocs.io/en/main/token.html
2. Скачайте расширение для браузера по ссылке на странице
3. Установите расширение в браузер
4. Откройте Яндекс.Музыку и авторизуйтесь
5. Нажмите на иконку расширения - скопируется токен

## Использование токена

Токен уже установлен в файле `youtube-audio-server/yandex_token.txt`

Текущий токен: `y0__xCpqe6YBxje-AYgpYjh0xYwuJyOpwgJTj9hBFevWOB5p2Y1uyvRZZ4z4A`

Пользователь: `monty.dark`

## Поиск музыки

1. Запустите приложение
2. Нажмите на кнопку Яндекс.Музыки в боковой панели (желтая иконка)
3. Введите запрос в поиск
4. Треки будут найдены в Яндекс.Музыке

## Обновление токена

Если токен перестал работать:

1. Получите новый токен через расширение
2. Отправьте POST запрос:

```bash
curl -X POST http://192.168.1.120:3000/api/yandex/auth \
  -H "Content-Type: application/json" \
  -d '{"token": "ВАШ_НОВЫЙ_ТОКЕН"}'
```

Или просто замените токен в файле `youtube-audio-server/yandex_token.txt` и перезапустите сервер.
