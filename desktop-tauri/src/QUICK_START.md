# 🚀 Быстрый старт Desktop версии

## Шаг 1: Установка зависимостей

```bash
cd desktop
npm install
```

## Шаг 2: Запуск сервера

В отдельном терминале:

```bash
cd ../youtube-audio-server
npm start
```

Должно появиться:
```
🎵 Music Streaming Server running on http://localhost:3000
📱 Your local IP: 192.168.1.120
```

## Шаг 3: Запуск desktop приложения

```bash
cd desktop
npm start
```

## Готово! 🎉

Приложение запустится с:
- Анимированным splash screen
- Glassmorphism дизайном
- Доступом ко всем источникам музыки

## Опционально: Яндекс.Музыка

Для доступа к "Моей волне" и полным трекам Яндекса:

1. Откройте music.yandex.ru
2. Войдите в аккаунт
3. F12 → Application → Cookies → music.yandex.ru
4. Скопируйте "Session_id"
5. Выполните:

```bash
curl -X POST http://192.168.1.120:3000/api/yandex/auth \
  -H "Content-Type: application/json" \
  -d '{"token":"ВАШТОКЕН"}'
```

Авторизация сохранится на сервере!

## Использование

### Моя волна
- Нажмите на карточку "Моя волна"
- Автоматически загрузятся персонализированные треки

### Поиск
- Нажмите "Поиск" в сайдбаре
- Введите запрос
- Выберите трек из результатов

### Массовое добавление
- Кнопка "+" в правом верхнем углу
- Введите список треков (по одному на строку)
- Нажмите "Добавить"

## Горячие клавиши

- `F12` - DevTools
- `F5` - Перезагрузка
- `Space` - Play/Pause
- `→` - Следующий трек
- `←` - Предыдущий трек

## Проблемы?

1. Проверьте что сервер запущен: http://192.168.1.120:3000
2. Откройте DevTools (F12) и посмотрите логи
3. Проверьте логи сервера в терминале

Enjoy! 🎵
