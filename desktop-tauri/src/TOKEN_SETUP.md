# Настройка токенов для музыкальных сервисов

## Яндекс.Музыка

### Получение Session_id токена:

1. Откройте браузер и зайдите на https://music.yandex.ru
2. Войдите в свой аккаунт
3. Откройте DevTools (F12)
4. Перейдите на вкладку **Application** (Chrome) или **Storage** (Firefox)
5. В левом меню выберите **Cookies** → `https://music.yandex.ru`
6. Найдите cookie с именем `Session_id`
7. Скопируйте его значение (длинная строка)

### Установка токена:

**Вариант 1: Через интерфейс приложения**
1. Откройте приложение Dotify
2. Нажмите на иконку настроек (шестеренка) в боковой панели
3. Вставьте токен в поле "Session_id токен"
4. Нажмите "Сохранить"

**Вариант 2: Через curl**
```bash
curl -X POST http://192.168.1.120:3000/api/yandex/auth \
  -H "Content-Type: application/json" \
  -d '{"token":"ВАШ_SESSION_ID_ТОКЕН"}'
```

## Spotify

### Получение Client ID и Client Secret:

1. Зайдите на https://developer.spotify.com/dashboard
2. Войдите в свой Spotify аккаунт
3. Нажмите **Create app**
4. Заполните форму:
   - **App name**: Dotify Desktop
   - **App description**: Music player
   - **Redirect URI**: `http://localhost:3000/callback`
   - Согласитесь с условиями
5. Нажмите **Create**
6. На странице приложения нажмите **Settings**
7. Скопируйте **Client ID**
8. Нажмите **View client secret** и скопируйте **Client Secret**

### Установка credentials:

**Вариант 1: Через интерфейс приложения**
1. Откройте приложение Dotify
2. Нажмите на иконку настроек (шестеренка) в боковой панели
3. Вставьте Client ID и Client Secret в соответствующие поля
4. Нажмите "Сохранить"

**Вариант 2: Через curl**
```bash
curl -X POST http://192.168.1.120:3000/api/spotify/auth \
  -H "Content-Type: application/json" \
  -d '{"clientId":"ВАШ_CLIENT_ID","clientSecret":"ВАШ_CLIENT_SECRET"}'
```

## Проверка статуса

После настройки токенов перезапустите сервер:

```bash
cd youtube-audio-server
node server.js
```

В логах вы должны увидеть:
```
✅ Yandex Music - Ready
✅ Spotify - Ready
```

## Выбор источника по умолчанию

В настройках приложения вы можете выбрать источник музыки по умолчанию:
- **Все источники** - поиск по всем доступным платформам
- **YouTube** - только YouTube
- **Яндекс.Музыка** - только Яндекс (требует токен)
- **Spotify** - только Spotify (требует credentials)
- **SoundCloud** - только SoundCloud

Выбранный источник будет использоваться для всех поисков, пока вы не измените его вручную через боковую панель или настройки.

---

## Примечания

- **Session_id** токен Яндекс.Музыки действителен несколько месяцев
- **Spotify credentials** не имеют срока действия
- Токены хранятся только в памяти сервера и сбрасываются при перезапуске
- Для постоянного хранения можно добавить токены в переменные окружения
