# Настройки WebGL шейдера волны

## Полный список настроек

### 🎨 Цвета
- **Цвет волны** - основной цвет волны
- **Цвет свечения** - цвет внутреннего и внешнего свечения
- **Цвет частиц** - цвет летающих частиц
- **6 пресетов**: По умолчанию, Фиолетовая, Зеленая, Красная, Голубая, Розовая

### 🎵 Реакция на звук
- **Реакция на бассы** (0-10x) - насколько сильно волна реагирует на низкие частоты
  - По умолчанию: 3.5x
  - Рекомендуется: 2-5x для умеренной реакции, 5-10x для экстремальной
  
- **Реакция на миды** (0-8x) - реакция на средние частоты
  - По умолчанию: 2.2x
  - Рекомендуется: 1.5-4x
  
- **Реакция на высокие** (0-5x) - реакция на высокие частоты
  - По умолчанию: 1.5x
  - Рекомендуется: 1-3x

### ✨ Эффекты
- **Свечение волны** - включает/выключает внутреннее и внешнее свечение
- **Отражения** - показывает отражение волны снизу
- **Частицы** - включает/выключает летающие частицы

### ⭐ Настройки частиц
- **Плотность частиц** (0-2) - количество частиц на экране
  - 0 = нет частиц
  - 1 = нормальная плотность (по умолчанию)
  - 2 = максимальная плотность
  
- **Размер частиц** (0.5-2) - размер отдельных частиц
  - 0.5 = маленькие
  - 1 = нормальные (по умолчанию)
  - 2 = большие

### 🌊 Настройки волны
- **Толщина волны** (0.5-2) - толщина линии волны
  - 0.5 = тонкая
  - 1 = нормальная (по умолчанию)
  - 2 = толстая
  
- **Интенсивность волны** (0.5-2) - амплитуда колебаний волны
  - 0.5 = спокойная
  - 1 = нормальная (по умолчанию)
  - 2 = бурная

## Как использовать

1. Открой **Настройки** в сайдбаре
2. Прокрути до секций с настройками волны
3. Настрой параметры в реальном времени
4. Все изменения сохраняются автоматически в localStorage

## Рекомендуемые пресеты

### Спокойная волна
- Реакция на бассы: 2x
- Реакция на миды: 1.5x
- Реакция на высокие: 1x
- Толщина: 0.8
- Интенсивность: 0.7

### Энергичная волна
- Реакция на бассы: 6x
- Реакция на миды: 4x
- Реакция на высокие: 2.5x
- Толщина: 1.2
- Интенсивность: 1.5

### Минималистичная
- Свечение: выкл
- Отражения: выкл
- Частицы: выкл
- Толщина: 0.6
- Плотность частиц: 0

### Максимальные эффекты
- Все эффекты: вкл
- Плотность частиц: 2
- Размер частиц: 1.5
- Реакция на бассы: 8x

## Производительность

Для слабых компьютеров рекомендуется:
- Отключить частицы
- Отключить отражения
- Уменьшить плотность частиц до 0.5
- Уменьшить реакцию на звук до 2x/1.5x/1x

## API для разработчиков

```javascript
// Изменить цвета
window.waveShader.setColors(
  { r: 0.7, g: 0.9, b: 1.0 }, // wave
  { r: 0.5, g: 0.7, b: 1.0 }, // glow
  { r: 1.0, g: 0.9, b: 0.4 }  // particle
);

// Применить пресет
window.waveShader.setColorPreset('purple');

// Настроить реакцию на звук
window.waveShader.setAudioReactivity(5.0, 3.0, 2.0);

// Включить/выключить эффекты
window.waveShader.setEffects(true, true, false); // glow, reflections, particles

// Настроить частицы
window.waveShader.setParticleSettings(1.5, 1.2); // density, size

// Настроить волну
window.waveShader.setWaveSettings(1.2, 1.5); // thickness, intensity
```
